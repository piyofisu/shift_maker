# 利用方法

# アプリケーションを作成した背景
前々職の同僚に業務上の課題をヒアリング。マネージャーに昇格していたことからシフトの作成を担当者になっていましたが、「シフト作成に時間がかかる。」とのことでした。
24時間営業でスタッフ人数も多く、考えなければいけない制約が多いので時間を取られてるようでしたので、シフト作成時間の短縮を目的に作成いたしました。
