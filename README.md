# アプリケーションを作成した背景
前々職の同僚に業務上の課題をヒアリング。「シフト作成に時間がかかる。」とのことでした。24時間営業でスタッフ人数も多く、考えなければいけない制約が多いので時間を取られていました。現在はExcelでシフトの作成をしているが、まずは同様の機能を持つシステムを作り、使ってもらい要望を聞きながら改良していこうと思い作成を決めました。
